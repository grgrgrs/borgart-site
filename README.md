# Borgart — Practical Data & AI for Small Business

Borgart exists for one purpose:  
**Helping small businesses see their data, understand it, and act on it.**

You don’t need a data team.  
You don’t need dashboards for dashboards’ sake.  
You need clarity → insight → action.

This site introduces the service framework behind that work.

---

## 🌱 What this project contains

| Area | Purpose |
|---|---|
| `/src/pages` | Main site pages including Home & Services |
| `/src/components` | Reusable UI blocks (ServiceCard, CTA, etc.) |
| `/src/layouts` | Base layout & site-wide structure |
| `global.css` | Core typography & spacing system |
| `tailwind.config.js` | Theme, breakpoints, design tokens |

Built with:

- **Astro 4** (framework)
- **TailwindCSS** (design system)
- **No runtime JS required** for core pages
- Fully responsive — fast mobile-first reading
